# Focus Management

### Question 1 : What is deep work?
- it is focused and distraction free work
- using deep work we can learn much faster
- it is essential for learning complex things
- we need to avoid social media and unnecessary emails
- we need to focus deeply without multitasking

### Question 2 : According to author of Deep Work Book, how to do deep work properly, in a few points?
- we need to plan specific time intervals for deep work
- we need to train ourselves to avoid checking phone and unnecessary emails
- use only tools that makes my life better and which doesn't distract deep work

### Question 3: How can you implement the principles in your day to day life?
- Schedule and restrict distractions
    - while doing deep work note every distraction, then try to avoid next time 
- Make it a habit - Schedule it in (1hrs)
    - we need to schedule a time for deep work , and schedule it and do in that interval only.
- Daily shutdown  - Schedule next day
    - at the evening times, we need to note the tasks which are incomplete and schedule it for next day
    - this makes evening shutdown freely and focus on next day
 
### Question 4 : What are the dangers of social media, in brief?
- social media makes it harder to focus on work
- it can distract from our work
- we think relax for 5 minutes using social media but it takes more than 1hr to stop using social media
- it makes us to feel lonely
- it makes human lazy
